File Access

MAT format

MATLAB工作区里的数据可以以 .mat 格式保存在文件中，使用 save 函数将数据存入文件，使用 load 函数从文件中读取数据

  • save 函数的语法如下：
    • save(filename， variables) 将变量 variables 以二进制形式存入文件中
    • save(filename, variables, '-ascii') 将变量以文本形式存入文件中
  • load 函数的语法如下：
    • load(filename)从二进制形式文件中读取数据.
    • load(filename,'-ascii')从文本形式文件中读取数据.

其中参数filename和variables都是字符串格式,若不指定variables参数,则将当前工作区内所有变量存入文件中.

复杂的数据格式,如struct和cell,不支持以二进制格式存储.

The explanation of MATLAB:

The size of the indicated variable or array appears to be changing with each loop iteration.Commonly, this message appears because an array is growing by assignment or concatenation.Growing an array by assignment or concatenation can be expensive.For large arrays,MATLAB must allocate a new block of memory and copy the older array contents to the new array as it makes each assignment.

Programs that change the size of a variable in this way can spend most of their run time in this inefficient activity.There is also significant overhead in shrinking an array on each iteration or in changing the size of a variable on each iteration. In such cases,MATLAB must reallocate and copy the contents repeatedly.

The use of disp

disp() 仅接受一个参数，下面的这种写法是错误的：

disp('aaa',32)

disp(['aaa',78]) 会输出’aaaN’，因为disp将78当做一个ASCII码字符作为显示

若想实现字符串和数字的混合输出，可以使用如下形式

disp(['aaa',num2str(32)])，输出aaa32。

num2str() 函数将数字转换为字符类型数组，再通过矩阵的拼接，将两个字符类型数组拼接位一个数组，从而正常显示输出

[!warning] 以下表达也可显示，但是显示形式不便于阅读

disp({'aaa',32})：使用元胞数组，显示的内容为 {'aaa'} {[32]}

The difference between sort and sortrows:

sort函数会将每一个row的内部进行排序后输出，而sortrows函数则只会比较第一个row，它将每一个column视为整体进行排序：

\mathrm{sort}(M) = \left(\begin{array}{cccc} 4 & 2 & 3 & 1\ 5 & 7 & 6 & 8\ 9 & 11 & 10 & 12\ 16 & 14 & 15 & 13 \end{array}\right)\\

多项式

使用MATLAB表示多项式（Polynomials）

在MATLAB中，多项式可以用向量表示（row vectors），第一位为多项式最高次项系数，最后一项为常数项

[!example]

$$
f(x) = x^{3}-2x-5
$$
可以用向量 p = [1 0 -2 -5] 表示

多项式求值：polyval()

使用 polyval(p,x) 计算多项式 p 在 x 每个点处的值

多项式的乘法：conv()

使用conv(p1, p2)函数可以对两个向量p1和p2进行卷积相乘,用于计算多项式的乘法.

例如多项式 $f(x) =(x^2+1)(2x+7)$ ，可用下式计算：

多项式的微分（derivative）：polyder()

使用 polyder(p) 函数可以计算多项式的导数

配合 polyvar() 可以计算出指定点的微分

多项式的积分（Integration）：polyint()

使用 polyint(p, k) 函数可以计算多项式p的积分,积分结果的常数项设为k。返回值为多项式的向量

k省略时常数项为0

[!example]
$$f(x) = 5x^4-2x^2+1$$

p = polyint([5 0 -2 0 1], 3)

得到 p = [1 0 -0.6667 0 1 3],表示计算得到如下积分:

$$\int f(x) dx =x^{5}-0.6667x^3+x+3$$
