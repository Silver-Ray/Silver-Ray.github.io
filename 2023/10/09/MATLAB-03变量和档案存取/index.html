Variables

Character(char)

  • A character is represented in ASCII using a numeric code between 0 to 255
  • Create a character or a string by putting them into a pair of apostrophe
  • Use uint16() to convert a char to its ASCII code

String

  • An array collects characters:
    • s1 = 'Example';
    • s2 = 'String'
  • String concatenation:
    • s3 = [s1 s2];
    • s4 = [s1; s2];
      • Error: Dimension of matries being concatenated are not consistent
      • s5 = [s1; s1]
        • It works well

Logic Operations and Assignments

Structure

  • A method of storing heterogeneous data
  • Structures contain arrays called fields
  • Students assignment grades:
函数名 函数作用
struct 创建结构体
struct2cell 将结构体转换为元胞数组
cell2struct 将元胞数组转换为结构体
isstruct 判断某变量是否是结构体
structfun 对结构体的每个字段都应用某函数
fieldnames 获取结构体的所有字段名
isfield 判断结构体是否包含某字段
getfield 获取结构体某字段的值
setfield 为结构体中的某字段赋值
rmfield 删除结构体中的某字段
orderfields 为结构体字段排序

[!example]

rmfield(student,'id')

Nesting Structure

Cell Array

  • Another method of storing heterogeneous data
  • Similar to matrix but each entry contains different type of data
  • Declared using { }
  • The two following expressions are the same
    • A{m, n} = element;
    • A(m ,n) ={element};
  • Each entry in a cell array hold s a pointer to a data structure
  • Different cells of the same cell array can point to different types of data structures

Access Cell Array

  • 有两种方式访问元胞数组中的数据,分别是: 单元索引()，和数组索引{}
    • 使用单元索引(),我们得到的是一个子元胞数组.
    • 使用内容索引{},我们得到的是元胞数组对应区域中的内容.

File Access

MAT format

MATLAB工作区里的数据可以以 .mat 格式保存在文件中，使用 save 函数将数据存入文件，使用 load 函数从文件中读取数据

  • save 函数的语法如下：
    • save(filename， variables) 将变量 variables 以二进制形式存入文件中
    • save(filename, variables, '-ascii') 将变量以文本形式存入文件中
  • load 函数的语法如下：
    • load(filename)从二进制形式文件中读取数据.
    • load(filename,'-ascii')从文本形式文件中读取数据.

其中参数filename和variables都是字符串格式,若不指定variables参数,则将当前工作区内所有变量存入文件中.

复杂的数据格式,如struct和cell,不支持以二进制格式存储.

Excel File

  • 读取Excel文件：[num,txt,raw] = xlsread(filename,sheet,xlRange)
    • 自动去掉文字部分
  • 写入Excel的语法: xlswrite(filename,A,sheet,xlRange)
  • 同时读取文本和数字可以使用以下操作：[Score Header] = xlsread('score.xlsx')
