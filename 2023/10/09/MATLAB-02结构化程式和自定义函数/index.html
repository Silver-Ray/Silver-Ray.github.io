MATLAB Script

  • A file containing a series of MATLAB commands
  • Pretty much like a C/C++ program
    • However Script does not need to be compiled before run
  • Scripts need to be saved to as <file>.m file before they can be run

%% is the symbol of section, we can run a single section

use start indent to make code more readable

Script Flow

  • Typically scripts run from the first line to the last
  • Structured programming techniques(subroutine, loop. condition, etc.) are applied to make the program looks neat

The explanation of MATLAB:

The size of the indicated variable or array appears to be changing with each loop iteration.Commonly, this message appears because an array is growing by assignment or concatenation.Growing an array by assignment or concatenation can be expensive.For large arrays,MATLAB must allocate a new block of memory and copy the older array contents to the new array as it makes each assignment.

Programs that change the size of a variable in this way can spend most of their run time in this inefficient activity.There is also significant overhead in shrinking an array on each iteration or in changing the size of a variable on each iteration. In such cases,MATLAB must reallocate and copy the contents repeatedly.

Tips for Script Writing

  • At the beginning of your script, use command
    • clear all to remove previous variables
    • close all to close all figures
  • Use semicolon ; at the end of commands to inhibit unwanted output
  • Use ellipsis ... to make scripts more readable
    • MATLAB2020 and later version can break line with no need to use ellipsis
  • Press Ctrl+C to terminate the script before
  • conclusion

The use of disp

disp() 仅接受一个参数，下面的这种写法是错误的：

disp('aaa',32)

disp(['aaa',78]) 会输出’aaaN’，因为disp将78当做一个ASCII码字符作为显示

若想实现字符串和数字的混合输出，可以使用如下形式

disp(['aaa',num2str(32)])，输出aaa32。

num2str() 函数将数字转换为字符类型数组，再通过矩阵的拼接，将两个字符类型数组拼接位一个数组，从而正常显示输出

[!warning] 以下表达也可显示，但是显示形式不便于阅读

disp({'aaa',32})：使用元胞数组，显示的内容为 {'aaa'} {[32]}

Variables Meaning

mfilename File name of currently running function
nargin Number of function input arguments
varargin Variable length of argrument list
varargout Varible length output argument list
